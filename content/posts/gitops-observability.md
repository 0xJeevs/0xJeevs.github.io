### Project Overview

Provisioned and deployed an automated, cloud-native GitOps CI/CD and observability platform focused on declarative, drift-free environment management and continuous delivery.

### Technical Implementation

- **Infrastructure as Code:** Provisioned Kubernetes infrastructure using **Terraform** and **Helm** for declarative, drift-free environment management.
- **Automated CI/CD:** Built automated CI/CD pipelines with **Jenkins** and **Docker** for testing, containerization, and secure artifact deployment.
- **GitOps Workflow:** Implemented GitOps continuous deployment using **ArgoCD** for automated synchronization, self-healing, and instant rollbacks.
- **Observability & Monitoring:** Integrated **Prometheus** and **Grafana** for metrics collection, real-time cluster monitoring, and system visibility.
- **Tech Stack:** Terraform, Kubernetes, ArgoCD, Jenkins, Docker, Python (FastAPI), Helm, Prometheus, Grafana.
