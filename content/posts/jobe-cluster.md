### Project Overview

Deployed and managed a distributed Jobe cluster designed for scalable, secure online code compilation and programming submission evaluation.

### Technical Implementation

- **Distributed Architecture:** Deployed and managed a distributed Jobe cluster configured with 1 Master node and 4 Worker nodes for scalable execution.
- **Load Balancing & Scheduling:** Configured job scheduling, worker load balancing, and sandboxed execution for secure evaluation of programming submissions.
- **Tech Stack:** Linux, Jobe, Bash, Networking, Distributed Systems, GCC/G++, System Administration.
